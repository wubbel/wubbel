HTML

HTML ist eine so genannte "seitenbeschreibende" Sprache. Sie enthält im Gegensatz zu Programmiersprachen keine Logik sondern beschreibt lediglich dein Aufbau einer Website. Man kann sich das vorstellen wie einen Bauplan eines Architekten für ein Haus. Mit HTML legt man fest, dass eine Website zum Beispiel einen Header hat. Darin befindet sich ein Logo und eine Navigation. Darunter befindet sich ein Container, der eine Überschrift sowie Textabsätze enthält und so weiter.

HTML gibt es schon seit einigen Jahren und hat sich im Lauf der Zeit weiterentwickelt. Aktuell verwenden wir den Standard 5. Du wirst da vielleicht auch öfter über den Begriff "HTML 5" drüber stolpern. Mit dieser Version von HTML sind einige neue HTML Elemente hinzugekommen und der Aufbau von HTML Seiten hat sich vereinfacht.

Struktur eines HTML Dokuments

Ein HTML Dokument benötigt immer ein Grundgerüst von drei Elemente:

  • html
  • head
  • body
